# Project Elliott: 엘리어트 파동 전문 매매 페르소나 및 트레이딩 봇

## 1. 프로젝트 정의 및 Oracle 세계관 내 위치
Project Elliott는 거대한 Project Oracle 생태계(다중 페르소나 자율 투자 분석 시스템) 내에서 동작하는 **최상위 기술적 분석 전문 서브 프로젝트**입니다. 
단순한 텍스트 지식 수집을 넘어, 매억남 등 실전 트레이더의 노하우를 학습하고 향후 **실제 비트코인 파동 트레이딩을 수행하는 오라클의 핵심 매매 페르소나(Expert Persona)**로 발전하는 것을 최종 목표로 합니다.

## 2. 오라클 아키텍처와의 융합 비전
- **Layer 2 (Brain - Persona)**: 오라클의 여러 전문가 위원회 중 '차트/파동 분석가' 역할을 담당합니다. 거시경제 분석 페르소나가 펀더멘털을 논할 때, Project Elliott는 타점과 파동의 위치를 근거로 토론에 참여합니다.
- **Layer 4 & 5 (Simulation & Decision)**: 최종적으로 비트코인 실시간 차트를 분석하여 파동 카운팅을 수행하고, 진입/청산 시그널을 생성하는 자동 매매 엔진으로 동작합니다.

---

## 3. 구현 로드맵 (Phases)

### Phase 1: 실전 트레이더 아카이브 추출 (L1 Ingestion)
- **목표**: 매억남 등 실전 엘리어트 파동 트레이더의 라이브 방송 영상(약 200~350여 개)을 처리하여 트레이딩 철학과 파동 카운팅 패턴을 추출.
- **과제**: 잡담 및 음악 재생 구간(영상 초반 0~7분 등)을 필터링하고, 오라클의 온톨로지(지식 그래프) 규격에 맞게 파동의 추론 과정(Reasoning Chain)을 정형화하여 `knowledge/` 폴더에 적재.

### Phase 2: 시계열 동기화 및 가격 앵커링 (Price Anchoring)
- **목표**: 차트 분석의 핵심인 '특정 캔들(봉)'과 발언의 싱크를 맞춤.
- **구현**: 영상의 발화 타임스탬프와 Binance OHLCV API(분봉/시간봉/일봉)를 결합. "여기서 반등했죠", "이전 저점"과 같은 모호한 지시대명사(Deictic expressions)를 실제 가격 데이터($73,300 등)로 치환.

### Phase 3: 매매 논리 평가 (Eval Framework 연동)
- **목표**: 학습한 파동 카운팅 시나리오와 실제 과거 차트의 결과를 대조.
- **구현**: 오라클의 Eval 파이프라인을 활용하여, 추출된 매매 논리가 실제로 유의미한 승률과 손익비를 보였는지(유효한 무효화 지점 설정이었는지) 백테스팅 및 수치화.

### Phase 4: 실시간 자동 매매 봇 구축 (최종)
- **목표**: 비트코인 및 주요 가상자산에 대한 실시간 엘리어트 파동 트레이딩 봇 완성.
- **구현**: 실시간 API 스트림을 받아 Project Elliott가 엘리어트 파동 카운팅을 지속적으로 업데이트하며, 오라클의 Layer 5(Decision)와 결합하여 포지션 진입/청산 실행.
