`src/discover.py`에 `get_popular_videos_by_stats()`가 이미 구현되어 있었으나 CLI에 연결되지 않은 상태.

**해결**:

- `--popular-scan` 플래그 추가 (기존 `--popular` 동작 유지)

### 7. POPULAR.md 링크 갱신

**문제**: `run_popular()`가 POPULAR.md 생성 시 채널 루트(`channel_dir`)만 탐색.
실제 파일은 `인기/` 서브디렉터리에 저장되므로 링크가 모두 YouTube URL로 폴백됨.

**해결**:
- `인기/` 서브디렉터리를 우선 탐색, 없을 시 채널 루트 탐색
- `os.path.basename()` → `os.path.relpath(channel_dir)` 로 상대경로 정확히 생성
- `import glob` 루프 외부로 이동 (반복 import 제거)

```python
# Before
matches = glob.glob(os.path.join(channel_dir, f"{pub}_{slug}*.md"))
rel = os.path.basename(matches[0])   # 인기/ 파일 못 찾음

# After
matches = (
    glob.glob(os.path.join(inki_dir, f"{pub}_{slug}*.md")) or   # 인기/ 우선
    glob.glob(os.path.join(channel_dir, f"{pub}_{slug}*.md"))    # 루트 폴백
)
rel = os.path.relpath(matches[0], channel_dir).replace(os.sep, '/')
```
